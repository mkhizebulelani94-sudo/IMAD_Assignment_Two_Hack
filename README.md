# IMAD_Assignment_Two_Hack
The name of this app is: Hack Spotter.

This app is a quiz based application with multiple screens and a review page at the end of it. To be more specific it is a cricket based quiz app made to put your cricketing knowledge to the test. All you have to do is say whether it is a myth or hack. It has 7 screens (5 questions) as you will get a total score out of 5 at the end of it. You will also get an automated response relative to the score that you will receive.
